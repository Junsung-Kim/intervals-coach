# 지식기반 (Knowledge base)

LLM 코칭 에이전트와 사람이 같은 용어·이론 프레임을 쓰도록 정리한 문서다. **의학적 조언이 아니다.** 부상·질환은 전문의·현장 코치에게 문의한다.

| 문서 | 내용 |
|------|------|
| [overview.md](overview.md) | CTL/ATL/TSB, wellness CSV, JSON 필드 개요 |
| [swimming.md](swimming.md) | 수영 역치(CSS), Intervals 매핑, 한계 |
| [running.md](running.md) | T-pace, 존, 러닝 데이터 전제 |
| [cycling.md](cycling.md) | FTP, 파워, NP, 파워미터 |
| [theories.md](theories.md) | 철학적 기반: Seiler(폴라라이즈드), Friel(주기화), Coggan(파워·CTL) |

`prompt.example.txt` 및 개인 `data/prompt.txt`에서는 이 디렉터리를 **근거·용어** 출처로 참조하도록 두는 것을 권장한다.
